Benchmarking a computationally demanding function in R and C++

The Fibonacci sequence mathematically defined as \[F_n = \begin{cases} n, & n\le 2\\ F_{n-1} + F_{n-2}, & n>2 \end{cases}\]

R implementation

f <- function(n){
  if (n < 2) return(n)
  return(f(n - 1)+f(n - 2))
}


(scores <- benchmark(f(15), f(20), f(25))[,1:4])
##    test replications elapsed relative
## 1 f(15)          100    0.07    1.000
## 2 f(20)          100    0.83   11.857
## 3 f(25)          100    8.76  125.143
scores %>% 
  ggplot(aes(test,relative, group = 1))+
  geom_line()+theme_minimal()+
  labs(title = "")
